## 📖 About the Project

University life often brings irregular schedules, late-night study sessions, and high caffeine consumption, leading to poor sleep quality. This project aims to quantify exactly how daytime habits impact nighttime rest. 

By feeding historical lifestyle data into a Machine Learning model, this application provides an accurate prediction of how many hours of sleep a student will get on a given night. It serves as an analytical tool to help students manage their time, stress, and health more effectively.

---

## 🧠 Machine Learning Concepts Used

This project heavily relies on fundamental Data Science and Machine Learning principles:

* **Supervised Learning:** The model does not guess blindly; it is trained on a labeled dataset (`data.csv`) where the historical outcomes (actual sleep hours) are provided alongside the input features.
* **Multiple Linear Regression:** Because we are predicting a continuous numerical value (hours of sleep) based on multiple independent variables, Multiple Linear Regression is utilized. The model calculates the optimal weights for the following equation:
  $$y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \dots + \beta_n x_n$$
* **Feature Selection & Analysis:** By analyzing the coefficients generated by the `scikit-learn` model, we determine which features strongly correlate with sleep loss (e.g., high caffeine) versus sleep gain.
* **Model Serialization:** The `pickle` library is used to save the trained model's state into a `.pkl` file. This allows the prediction script to run instantly without needing to retrain the algorithm every time.

## 📊 Dataset Features

The model processes five distinct continuous and discrete variables:
1.  **`StudyHours`**: Total hours dedicated to academic work during the day.
2.  **`CaffeineCups`**: Number of caffeinated beverages consumed.
3.  **`ScreenTimeHours`**: Non-academic screen time (phones, tablets, gaming).
4.  **`Deadlines`**: Number of assignments or exams due within the next 48 hours (measuring stress/urgency).
5.  **`ExerciseHours`**: Time spent performing physical activities.

*(Target Variable to predict: **`SleepHours`**)*
